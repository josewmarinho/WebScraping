# :construction_worker: Como rodar
```bash
# Clone o Repositorio
$ git clone https://github.com/josewmarinho/bot-dados.git
```

### 💻 Rode o Projeto Web

```bash
# Vá para a pasta src
$ cd src

# Instale as depedencias
$ npm i cheerio request request-promise json2csv

# Rode o roteiro
$ node index.js
```
